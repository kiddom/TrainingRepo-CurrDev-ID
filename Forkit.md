# Fork It! Exercise

This is a quick exercise to practice **forking** a repository and **creating a branch** in GitHub. We're not writing any code — just editing the practice block below.

## How to Fork and Branch in GitHub

1. **Fork** this repository to your own GitHub account.
   - Click the **Fork** button in the top-right of the repo page.
   - Confirm by clicking **Create fork**. You now have your own copy under your account.
2. In your fork, **create a new branch**.
   - Click the branch dropdown (usually says `main`).
   - Type a new branch name like `your-name/forkit-practice` and click **Create branch**.
3. Open this file (`ManualBranch.md`) in your new branch and click the pencil icon to edit.
4. **Overwrite** the practice block below with your own content — a fun fact, a favorite subject, a quick sentence, anything.
5. **Commit** your change to your branch with a short commit message.
6. Open a **Pull Request** from your branch back to the original repo's `main`.
